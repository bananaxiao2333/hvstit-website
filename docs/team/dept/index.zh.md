# 部门

## 综合与组织部门

!!! info inline end ""

    <figure markdown="span" style="max-width: 300px;">
    ![综合与组织部门](gene.jpg){ loading=lazy }
    <figcaption>综合与组织部门</figcaption>
    </figure>

综合与组织部门是团队的协调枢纽，负责团队日常运营、资源统筹与项目进度管理。部门成员负责制定工作计划、协调跨部门协作、管理项目资源，确保团队高效运转。同时承担对外联络与文档管理工作，是团队实现目标的重要保障。
## 理论与探索部门

!!! info inline end ""

    <figure markdown="span" style="max-width: 300px;">
    ![理论与探索部门](theo.jpg){ loading=lazy }
    <figcaption>理论与探索部门</figcaption>

理论与探索部门是团队的技术智库，专注于前沿技术研究与理论模型构建。部门成员负责开展可行性研究、技术方案设计、仿真模拟与理论验证工作。通过对航空航天、智能系统、光计算等领域的深入探索，为团队项目提供坚实的技术理论基础与创新方向指引。
## 化合与材料部门

!!! info inline end ""

    <figure markdown="span" style="max-width: 300px;">
    ![化合与材料部门](chem.jpg){ loading=lazy }
    <figcaption>化合与材料部门</figcaption>
    </figure>

化合与材料部门负责推进剂配方研发、材料测试与性能优化工作。部门成员专注于燃料化学研究、材料耐热性测试、推进剂性能参数优化，确保火箭发动机系统的安全可靠运行。同时开展新材料实验，探索更高效、环保的推进方案，为航天工程提供关键材料支撑。
## 电子与信息部门

!!! info inline end ""

    <figure markdown="span" style="max-width: 300px;">
    ![电子与信息部门](elect.jpg){ loading=lazy }
    <figcaption>电子与信息部门</figcaption>
    </figure>

电子与信息部门是团队的技术核心，负责电子系统设计、嵌入式开发与信息系统建设。部门成员专注于飞行控制系统、传感器网络、数据采集与处理平台的开发，确保火箭项目的精准控制与智慧校园系统的稳定运行。通过持续技术创新，为团队提供可靠的电子与信息技术解决方案。
## 工程与组装部门

!!! info inline end ""

    <figure markdown="span" style="max-width: 300px;">
    ![工程与组装部门](engin.jpg){ loading=lazy }
    <figcaption>工程与组装部门</figcaption>

工程与组装部门负责将设计方案转化为实体成果，承担机械设计、部件制造与系统集成的重任。部门成员负责火箭结构设计、3D打印制造、部件精密加工与整体组装调试，确保设计方案的高质量实现。通过严格的工程标准与精湛的工艺，为团队项目提供可靠的技术支撑。

道德与伦理监察委员会是团队的质量与安全守护者，负责确保项目符合科学伦理、安全标准与工程规范。委员会成员负责制定操作规程、监督实验安全、评估项目风险，确保火箭发射、化学实验等工作在安全可控的范围内进行。通过严格的质量管控与伦理监督，保障团队成员安全与项目合规性。

# 维科创组主页

!!! info inline end ""

    ![](assets/logo.png){ loading=lazy }

欢迎访问维港科技创新小组（VHSTIT）的官方主页。我们是直辖于暨大港澳子弟学校（Affiliated School of JNU for Hong Kong and Macao's Students，简称ASJ）校级的科技创新团队，致力于通过跨学科协作与前沿技术研究推动科技创新与学术进步。


## 核心使命与愿景

维科创组秉承"技术驱动创新，实践成就未来"的理念，专注于解决实际工程问题，培养成员的创新思维与工程实践能力。我们致力于构建一个开放、协作的科技创新平台，为校园科技创新文化建设贡献力量。

## 当前重点项目

目前，我们正在积极推进以下三大核心项目：

1. [**火箭发射项目**](projects/rocket/index.md)：一项开创性航空航天工程项目，旨在为小规模航空航天应用开发可持续且高效的推进系统，实现精准定高飞行与全景影像拍摄。
2. [**智慧校园系统**](../projects/smart-school)：基于物联网和人工智能技术的综合平台，旨在优化校园运营效率，提升校园智能化管理水平。
3. [**光纤计算机**](../projects/computer)：一项突破性计算技术研究，专注于利用光学计算原理实现前所未有的数据处理速度与能效比。

## 团队构成

我们的团队由五名充满热情且具备专业背景的成员组成，每位成员在各自领域拥有独特专长，共同构建了一个跨学科的技术创新团队。团队成员分工明确，协作高效，确保项目顺利推进。

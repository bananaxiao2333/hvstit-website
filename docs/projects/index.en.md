# Project Overview

## Project System Introduction

The Victoria Harbour Science and Technology Innovation Team (VHSTIT) currently maintains three core project systems covering aerospace, intelligent systems, and cutting-edge computing technology fields. Each project has clear technical objectives, implementation pathways, and expected outcomes, demonstrating the team's comprehensive capabilities in interdisciplinary technological innovation.

## Project Classification System

<div class="grid cards" markdown>

-   :material-rocket:{ .lg .middle } __Rocket Launch Project__

    ---

    Focused on small-scale aerospace technology development, achieving precise altitude control flight through OP-25 engines, and completing campus aerial imaging with panoramic cameras, filling the gap in campus aerospace engineering practice.

    [:octicons-arrow-right-24: Project Details](rocket/index.md)

-   :material-school:{ .lg .middle } __Smart Campus System__

    ---

    Based on IoT sensor networks and AI algorithms, building an intelligent campus management platform to achieve core functions such as facility monitoring, energy optimization, and safety alerts, enhancing campus operational efficiency.

    [:octicons-arrow-right-24: Project Details](smart-school/index.md)

-   :material-vanity-light:{ .lg .middle } __Fiber-Optic Computer Research__

    ---

    Exploring cutting-edge optical computing technology, researching data processing solutions based on photon transmission, aiming to break through traditional electronic computing bottlenecks and achieve computing systems with higher energy efficiency and processing speeds.

    [:octicons-arrow-right-24: Project Details](computer/index.md)

</div>

## Project Management Principles

All projects follow the implementation principle of "technically feasible, clearly targeted, and resource-controllable," ensuring each project has a clear technical roadmap and measurable phased objectives. Project teams conduct regular technical reviews and progress assessments to ensure projects advance according to plan.

# 项目总览

## 项目体系介绍

维港科技创新小组（VHSTIT）当前构建了三大核心项目体系，涵盖航空航天、智能系统与前沿计算技术领域。各项目均具备明确的技术目标、实施路径与预期成果，体现了团队在跨学科技术创新方面的综合能力。

## 项目分类体系

<div class="grid cards" markdown>

-   :material-rocket:{ .lg .middle } __火箭发射项目__

    ---

    专注于小型航空航天技术研发，通过OP-25型发动机实现精准定高飞行，搭载全景相机完成校园高空影像采集，填补校园航天工程实践空白。

    [:octicons-arrow-right-24: 项目详情](rocket/index.md)

-   :material-school:{ .lg .middle } __智慧校园系统__

    ---

    基于物联网传感网络与人工智能算法，构建校园智能化管理平台，实现设施监控、能耗优化、安全预警等核心功能，提升校园运营效率。

    [:octicons-arrow-right-24: 项目详情](smart-school/index.md)

-   :material-vanity-light:{ .lg .middle } __光纤计算机研究__

    ---

    探索光学计算技术前沿，研究基于光子传输的数据处理方案，旨在突破传统电子计算瓶颈，实现更高能效比与处理速度的计算系统。

    [:octicons-arrow-right-24: 项目详情](computer/index.md)

</div>

## 项目管理原则

所有项目均遵循"技术可行、目标明确、资源可控"的实施原则，确保每个项目具备清晰的技术路线图与可衡量的阶段性目标。项目团队定期进行技术评审与进度评估，确保项目按计划推进。
